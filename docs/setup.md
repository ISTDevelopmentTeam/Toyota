## Requirements

- PHP >= 8.2
- Composer
- Node.js and npm
- MySQL
- XAMPP

## Installation Steps

1. Clone the repo
2. Install PHP dependencies using Composer
3. Set up `.env` file and configure database
4. Run migrations
5. Install and compile frontend assets using npm
6. Serve the app with `php artisan serve`
